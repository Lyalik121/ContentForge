# ContentForge
AI platform for content repurposing
 
AI-платформа для переробки та створення контенту. З одного запису
(відео, подкаст, лекція) генерує готові пости для Instagram, TikTok,
LinkedIn і Telegram. Також уміє створювати контент-плани та сценарії
з нуля за заданою темою.

## Технології
- Backend: Go (Fiber)
- База даних: MS SQL
- AI: Whisper API (транскрипція), Gemini 1.5 Flash (генерація)
- Frontend: React / Next.js
- Інфраструктура: Docker, Railway
